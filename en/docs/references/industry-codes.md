> 来源：https://developer.work.weixin.qq.com/document/path/90314

## API Usage Scenarios

行业和规模信息在以下场景使用：

1. **创建企业**：注册时选择行业和规模
2. **获取企业信息**：`GET /cgi-bin/corp/get_info` 返回 `industry` 和 `sub_industry`
3. **通讯录同步**：企业信息同步时包含行业分类
4. **数据统计**：部分统计 API 按行业分类

> ⚠️ 行业代码可能随企业微信版本更新而新增，开发者在处理时应做好未知代码的兼容。
